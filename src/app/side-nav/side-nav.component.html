With this web application you can scan any EU Digital COVID Certificate emitted to extract and display all the information it encloses.

Loading in progress...

It is an open-source application and bug reports, suggestions, contributions, etc. are welcome.

No data whatsoever is stored, collected or shared: the processing of the certificate is entirely done locally within your browser.

This web application is in no way affiliated, associated, authorized, endorsed by, or in any way officially connected with any government agency.
